# Gearbox Design & Analysis Project

## Project Overview
This project involves 3D modeling and assembly of a gearbox using SolidWorks.

## Tools Used
- SolidWorks
- AutoCAD
- CATIA (Basic Modeling)

## Features
- 3D Part Modeling
- Assembly Design
- Exploded View
- Bill of Materials (BOM)
- Basic Stress Analysis (FEA)

## Objective
To understand mechanical power transmission and gear system design.
